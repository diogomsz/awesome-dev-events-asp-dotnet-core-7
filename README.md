
# Awesome Dev Events

Um projeto para fins educacionais e práticos de uma API Rest de eventos de programação




## Funcionalidades

- Cadastro
- Listagem
- Detalhes
- Atualização
- Remoção
- Cadastro de Palestrantes



## Tecnologias e Ferramentas
- Visual Studio 2022
- ASP.Net Core 7
- EF Core
- Swagger
